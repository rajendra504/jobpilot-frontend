Step-by-step guide

  1. Open Login — This opens the portal's login page in the server's headless browser.
  2. Complete Login on Server — Because the browser is on the server (not your screen), you need to wait ~30 seconds for it to load, then the server's Chromium will be sitting at the login page. If you are running locally with headless=false, a Chromium window will appear on your screen.
  3. Confirm & Save Session — Click this button after the server has logged in. The cookies are encrypted and stored in the database.
  4. Scrape Jobs — Go to Job Listings and click Scrape Jobs. The scraper will use the saved session automatically.
